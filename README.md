# AITestingRepository

AI Testing projects pushed to this repo.

## Contents

### `Chapter02_PromptEngineering/`

Prompt-engineering driven test automation projects.

| Project                                     | Description                                                                                              |
| ------------------------------------------- | -------------------------------------------------------------------------------------------------------- |
| `Project01_TC_generation_RICEPOT/`          | Test-case generation using the RICE-POT prompt framework.                                                |
| `Project02_SeleniumFramework_RICEPOT/`      | Advanced Selenium + TestNG Java framework (Salesforce login flows) built via RICE-POT prompting.         |
| `Project03_PlaywrightAPIFramework_RICEPOT/` | Advanced Playwright API framework (TypeScript) for the Restful-booker API, built via RICE-POT prompting. |

## Quick Start

### Java / Selenium project

```bash
cd Chapter02_PromptEngineering/Project02_SeleniumFramework_RICEPOT/AdvancedSeleniumFramework
mvn test
```

### TypeScript / Playwright API project

```bash
cd Chapter02_PromptEngineering/Project03_PlaywrightAPIFramework_RICEPOT/AdvancedPlaywrightFramework
npm install
npx playwright test
```
